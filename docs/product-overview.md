# Product Overview

Stay Loyal is a loyalty infrastructure platform for NFT communities.

The platform enables NFT projects to reward holders for long-term commitment through staking, loyalty scoring, progression systems, rewards, and NFT evolution.

The goal is to shift NFT communities away from speculation and toward sustainable engagement.

## Target Customers

- NFT Collections
- Gaming Projects
- Membership Communities
- DAOs
- Creator Communities

## Value Proposition

Reward loyalty instead of hype.
