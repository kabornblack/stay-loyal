# Stay Loyal

Stay Loyal is a Solana NFT loyalty infrastructure platform designed to help NFT communities reward long-term holders through staking, XP, ranks, rewards, and NFT evolution.

## What It Does

- NFT staking for supported Solana collections
- Loyalty XP based on staking duration
- Holder ranks such as Bronze, Silver, Gold, and Platinum
- NFT evolution progress system
- Founder dashboard for community analytics
- Rewards and mission structure for community engagement

## Tech Stack

- React
- TypeScript
- Vite
- TailwindCSS
- Supabase
- Solana Wallet Adapter
- Helius API
